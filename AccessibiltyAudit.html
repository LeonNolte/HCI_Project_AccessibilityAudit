Wave Summary Screen and Accessibility Issues

Pinterest uses the descriptive text as alternative text, which can produce long alternative texts and texts that do not accurately describe the image. If Pinterest does not find an image description, it uses the attribute text as alternative text, which gives a redundant title text warning. The redundant link warning relates to Pinterest offering links by clicking on the image and by clicking on the link below the image.

A screenshot of the main page with the WAVE accessibility tool turned on

Additionally, some embedded links to the website where the image is from can be empty.

A screenshot of the main page with the WAVE accessibility tool turned on

When clicking on an image, the comment section form is not described as such, producing an error.

Wave Structure Screen

Pinterest home page uses a one-layer structure, meaning that the entries are not sub-headings.

A screenshot of the main page with the WAVE accessibility tool turned on

On the profile, the structure adheres closely to the expected structure, listing the folders as subheadings.
